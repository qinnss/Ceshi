# 美式炒蛋制作指南

## 🍽️ 简介

美式炒蛋以松软、湿润、奶油般的口感著称，不同于中式炒蛋的焦香。本指南提供标准化制作流程。

## 📋 食材清单

| 材料         | 用量           | 备注         |
| ------------ | -------------- | ------------ |
| 鸡蛋         | 2个（大号）    | 室温更佳     |
| 无盐黄油     | 1汤匙（约15g） |              |
| 牛奶或淡奶油 | 1汤匙（可选）  | 增加奶香     |
| 盐           | 1/4茶匙        | 出锅前放     |
| 黑胡椒       | 适量           | 现磨风味更佳 |

## 🛠️ 所需工具

- 不粘平底锅
- 硅胶铲或木铲
- 搅拌碗
- 打蛋器或叉子

## 👨‍🍳 制作步骤

### 1. 准备蛋液

```markdown
1. 将鸡蛋打入碗中
2. 加入牛奶/奶油（如使用）
3. 用叉子轻轻打散（不要过度搅拌）
```

### 2. 烹饪过程

```markdown
1. **低温预热** - 中小火热锅
2. **融化黄油** - 锅底均匀覆盖
3. **倒入蛋液** - 立即开始搅拌
4. **持续推动** - 用铲子从外向内推
5. **离火时机** - 蛋液80%凝固时关火
6. **余温加热** - 用锅余温完成烹饪
```

### 3. 装盘调味

```markdown
1. 盛入预热过的盘子
2. 撒上盐和黑胡椒
3. 可加香葱或欧芹装饰
```

## 💡 成功秘诀

| 关键点       | 原因                   |
| ------------ | ---------------------- |
| **低温慢炒** | 避免蛋白质过度收缩变硬 |
| **持续搅拌** | 形成细腻的奶油状质地   |
| **及时离火** | 利用余温加热，防止过熟 |

## 🎨 风味变化

### 基础版

- 奶酪碎（出锅前加入）
- 炒香的培根丁

### 高级版

- 烟熏三文鱼 + 莳萝
- 炒蘑菇 + 帕玛森奶酪

## 📝 常见问题

**Q: 为什么我的炒蛋很干？**  
A: 火太大或烹饪时间过长。

**Q: 可以提前准备吗？**  
A: 不建议，美式炒蛋最佳风味是现做现吃。

**Q: 用普通锅可以吗？**  
A: 建议使用不粘锅，成功率更高。
